# 攪拌槽反應器互動式教學

這是一個可離線開啟的靜態教學網頁，用於介紹攪拌槽反應器（stirred tank reactor, STR）的構造、傳熱方式、批式操作、連續式操作與多槽串聯概念。

## 使用方式

直接開啟 `index.html` 即可使用，不需要安裝套件或啟動伺服器。

## 主要功能

- 可點選構造圖中的馬達、攪拌軸、攪拌器、夾套、盤管、進料、冷卻水、熱媒與卸料口。
- 左側提供六段式學習路徑，適合課堂講解或自學。
- 可切換單槽構造、批式操作濃度曲線、多槽串聯三種檢視。
- 透過操作模式、攪拌轉速、進料濃度與溫控強度滑桿，觀察混合均勻度、估計轉化率、溫度趨勢與操作風險。
- 內建快速檢核題，協助理解 CSTR 完全混合與轉化率的關係。

## 教學重點

攪拌槽反應器的核心假設是槽內充分混合，因此連續式 CSTR 的出口濃度可視為等於槽內濃度。批式操作中濃度會隨時間改變；連續式操作達穩態後濃度可近似維持不變。單一 CSTR 可能因進料立即稀釋而降低平均反應速率，多槽串聯可改善此限制。
